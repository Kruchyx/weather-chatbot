# Weather AI Assistant

Weather AI Assistant to prosty chatbot webowy napisany w HTML, CSS i JavaScript. Aplikacja analizuje dane wpisane przez użytkownika i rekomenduje odpowiedni ubiór do pogody.

## Funkcjonalności

- interfejs czatu
- dynamiczne odpowiedzi bota
- analiza słów kluczowych
- rekomendacja ubioru do pogody
- obsługa klawisza Enter
- animacja "Bot pisze..."
- responsywny wygląd
- nowoczesny interfejs typu glassmorphism
