# Weather AI Assistant

## Opis projektu

Weather AI Assistant to inteligentny chatbot webowy stworzony z wykorzystaniem HTML5, CSS3 oraz JavaScript. Aplikacja umożliwia użytkownikowi uzyskanie rekomendacji dotyczących ubioru na podstawie aktualnych warunków pogodowych.

Bot analizuje dane pogodowe pobierane z OpenWeather API oraz generuje odpowiednie sugestie dotyczące stroju, ochrony przed deszczem, zimnem oraz wysokimi temperaturami.

## Funkcjonalności

* Interaktywny interfejs czatu
* Dynamiczne odpowiedzi bota
* Pobieranie aktualnej pogody dla miasta
* Rekomendacje ubioru na podstawie temperatury i warunków pogodowych
* Obsługa klawisza Enter
* Animacja „Bot pisze...”
* Zapisywanie historii rozmów (LocalStorage)
* Responsywny design
* Nowoczesny interfejs typu Glassmorphism

## Architektura rozwiązania

Frontend (HTML/CSS)

JavaScript Chat Engine

OpenWeather API

Analiza danych pogodowych

## Przykładowa rozmowa

Użytkownik:
Warszawa

Bot:
Warszawa: 20°C, bezchmurnie.
Wystarczy lekka bluza albo koszulka z długim rękawem.

## Uruchomienie projektu

1. Sklonuj repozytorium.
2. Otwórz projekt w Visual Studio Code.
3. Uruchom plik index.html przy pomocy rozszerzenia Live Server.
4. Wprowadź nazwę miasta i sprawdź rekomendacje pogodowe.
